# eLUX Mobile Services
Unofficial mobile app for retrieving information about your eLUX SIM card. It uses the following built API https://github.com/hamzakat/elux-api 

## Prerequisites
- React Native
- Expo SDK
- React Native Paper UI

## Setup
```
yarn add
```

## Build
- Android
```
expo build:android
```
- iOS
```
expo build:ios
```
